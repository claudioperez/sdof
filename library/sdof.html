sdof.integrate(m, c, k, f, dt, u0=0.0, v0=0.0, out=None, fy: float | None = None, alpha_m: float = 1.0, alpha_f: float = 1.0, beta: float = 0.25, gamma: float = 0.5)#

This function integrates scalar differential equations of the form

\[m \ddot{u} + c \dot{u} + k u = f(t)\]

for constant coefficients \(m\), \(c\) and \(k\). Integration is carried out using a Generalized - \(\alpha\) integrator that is implemented under the hood in highly optimized multi-threaded C code.

This function is a wrapper around the C function sdof_integrate_unrolled.
